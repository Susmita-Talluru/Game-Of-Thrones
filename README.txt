This Angular web application contains all information about a popular TV Series Game of Thrones and its related book series.

It contains the following views:

1)View 1 - A home page with an introduction to GOT series and a navbar to list all books, characters and houses in the form of bootstrap cards. It also contains a search box to
                    filter the response data. Also the data is sorted in alphabetical order.

2)View 2 - On clicking the card, a view containing the detail of that book/character/house will open containing all the information about it in details.
